# SkinTwin Customer Portal

Multi-tenant B2B/B2B2C customer portal (React 19 + tRPC + Drizzle) for the SkinTwin platform: four user roles, consultations, bookings, orders, payments, and commissions. See [todo.md](./todo.md) for the feature checklist and [docs/CI.md](./docs/CI.md) / [docs/E2E_TESTING.md](./docs/E2E_TESTING.md) for CI and testing documentation.

## Ecosystem

This repo is part of the [SkinTwin-AI ecosystem](https://github.com/jax-a11y/skintwin-ecosystem-design), where it sits in the **business-apps** layer as the **customer-org-service** (Customer & Org Service: users, organizations, tenants, auth). See [ECOSYSTEM.md](./ECOSYSTEM.md) for its contracts, events, and service-discovery details.
